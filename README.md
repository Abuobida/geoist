* Gravity adjustment
* Gravity inversion
* Gravity modelling

## Features

* Our algorithms take advantages of Bayesian inversion framework.
* Build upon popular Python packages such as Numpy, Scipy, Pandas, Matplotlib etc. Well integrated into python ecosystem.
